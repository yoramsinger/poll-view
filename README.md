# מעקב סקרים – כנסת ישראל
אפליקציה אינטראקטיבית להצגת סקרי הכנסת לאורך זמן.

🔗 [**פתח אפליקציה**](https://yoramsinger.github.io/poll-view/)

---

## תכונות
### גרף אינטראקטיבי
- ציר X: זמן (תאריך הסקר)

- ציר Y: מספר מנדטים

- גרף נפרד לכל מפלגה × סוקר

- כל מפלגה בצבע קבוע; כל סוקר ב-dash שונה

- Hover מדגיש גרף אחד ומדהה את השאר

- Tooltip מציג תאריך, מנדטים, סוקר וערוץ תקשורת

### סינון מפלגות
בחירה / ביטול בחירה של מפלגות להצגה. כפתורי **הכל / נקה** לנוחות.

### סינון סוקרים
בחירה / ביטול בחירה של סוקרים. כפתורי **הכל / נקה** לנוחות.

### ממוצע שבועי
כפתור **⌀ ממוצע שבועי** בתחילת פס הסוקרים.

- כשמופעל — עבור כל מפלגה נבחרת מוצג **גרף יחיד** המייצג את הממוצע השבועי

- הממוצע מחושב **דינמית** לפי הסוקרים הנבחרים בלבד:

- לכל שבוע, לכל סוקר — מחושב ממוצע הסקרים שלו אותו שבוע (מונע ספירה כפולה של גרסאות שונות של אותו סקר)

- הממוצע הסופי הוא ממוצע על פני הסוקרים הנבחרים

- ניתן להוסיף/הסיר סוקרים כשהממוצע פעיל — הגרף מתעדכן מיד

- ה-tooltip מציג את מספר הסוקרים שנכנסו לחישוב

## נתונים
הנתונים נמצאים בקובץ `polls.csv` ומכסים סקרים מ-2026.

עמודות: מספר סקר, תאריך, משיבים, כלי תקשורת, עורך משאלים, ומנדטים לכל מפלגה.
